== Description ==

This plugins adds functionality to Elementor Pro.
Visit the [Repo](https://github.com/NicolaCavallazzi/elementor-super-cat) page on Github to know more.

== Installation ==

1. Upload the archieve (.zip) file from the "Add new Plugin" page in Wordpress
1. Activate the plugin through the 'Plugins' screen in WordPress
1. You will find the new widgets in the category "Super Cat" in Elementor editor
